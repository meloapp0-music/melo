# Canva Content Calendar — Melo

The Claude-writes / you-paste workflow. Pairs with
`marketing/social-calendar-week-by-week.md` (which uses your existing
assets). This file covers the NEW graphics you'll create in Canva.

**Cadence:** one Sunday batch session, ~45 min, makes a whole week.

**Output target:** 3 Canva designs per week max. More than that and
you'll burn out by week 2. Less than that and your feed feels dead.

---

## Step 1: Build the evergreen templates once (do this first)

These are templates you make ONCE in Canva and reuse forever with
swap-in copy. After this is set up, weekly designs become ~5 min each
because you're just replacing text.

### Template A — "Quote post" (1080×1080)
For Wrapped-season vibes / emotional copy.

- Background: photo of stage lights / crowd silhouette / concert dust
  (search Canva photos for "concert," "stage lights," "crowd")
- Dark gradient overlay (top to bottom: `#0A0F1C` 60% → `#0A0F1C`
  10%) — so text reads on any photo
- **Center text block** (Outfit Bold, 64-72px, white):
  big swappable headline
- **MeloIcon** bottom-left, 60×60px
- **`@melo.show`** bottom-right, Inter Medium 16px, white 70% opacity

### Template B — "Stat callout" (1080×1080)
For sharing user/feature numbers — App Store launch stats, milestones,
"we just shipped" moments.

- Solid `#0A0F1C` background
- **Huge orange number** (Outfit Black, 240px, `#E8573A`) centered
- **Label below** (Inter Bold, 24px, `#FBF6EE`) — e.g. "shows logged
  this week"
- Subtle radial gradient behind the number (faint `#E8573A` 8%)
- MeloLockup small at bottom

### Template C — "Story announcement" (1080×1920)
For Instagram Stories — feature drops, version releases, polls.

- Vertical orientation, full bleed
- Top third: dark cream background `#FBF6EE` with brown text
  `#3A2C20`
- Middle third: orange band `#E8573A` with white headline
- Bottom third: black `#0A0F1C` with CTA
- MeloIcon top-left

### Template D — "Feature card" (1200×675, Twitter card size)
For Twitter posts that link to the app. Twitter renders this as a
link preview.

- Left half: dark gradient background with **headline text** (Outfit
  Bold, 48px white)
- Right half: an iPhone mockup PNG (you can use one of the carousel
  slides as a starter)
- MeloLockup bottom-left, "melo.show" + App Store badge bottom-right

**Spend ~1 hour Sunday afternoon making these 4 templates.** Save
each one to your Canva folder "Melo / Evergreen." Then weekly work is
just text-swapping.

## Step 3: The Sunday workflow

Every Sunday, ~45 minutes:

1. **Open this doc.** Read this week's section.
2. **Open Canva** → your "Melo / Evergreen" folder → duplicate
   the template you need.
3. **Paste the headline + sub** from the brief. Brand Kit makes
   colors + fonts automatic.
4. **Export** as PNG (1080×1080 for square, 1080×1920 for stories,
   1200×675 for Twitter).
5. **Drop into Buffer** to schedule for the week's slots per the
   social calendar.
6. **Done.**

Total time per week: ~45 min if you're focused. Faster after you've
done it once.

## What NOT to make in Canva

To save your sanity, these are NOT Canva work:
- Reels / TikToks → CapCut + OpusClip
- App Store screenshots → already done, native iPhone screenshots
- App icon / favicon → already done, Looka logo
- In-app design → that's the React codebase, not graphics

Canva is just for static social graphics. Don't let it expand.
